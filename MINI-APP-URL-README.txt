Почему в Telegram не открывается Mini App (ERR_NAME_NOT_RESOLVED)
================================================================

В .env записан адрес туннеля (trycloudflare.com). Этот адрес РАБОТАЕТ только пока
запущено окно с туннелем. Как только туннель закрыли — адрес мёртвый.

ЧТО ДЕЛАТЬ КАЖДЫЙ РАЗ ПЕРЕД ИСПОЛЬЗОВАНИЕМ БОТА:
-------------------------------------------------
1. Запусти ZAPUSK-PROSTO.bat (или только туннель для Mini App на порту 3000).

2. В окне с заголовком "PadelSense - Tunnel App" через несколько секунд появится строка:
   https://ЧТО-ТО-СЛУЧАЙНОЕ.trycloudflare.com

3. Скопируй этот URL целиком.

4. Открой файл .env в корне проекта. Найди строку:
   MINI_APP_URL=
   Вставь скопированный URL:
   MINI_APP_URL=https://xxx.trycloudflare.com

5. Перезапусти бота (закрой окно "PadelSense - Bot" и снова открой его из ZAPUSK-PROSTO.bat,
   или запусти вручную: python -m bot.main).

6. В Telegram нажми «Открыть приложение» — должно открыться.

ВАЖНО: Окно "Tunnel App" НЕ ЗАКРЫВАЙ пока пользуешься ботом в Telegram.
Если закрыл туннель — адрес помер, нужно снова скопировать новый URL из нового запуска туннеля и обновить .env и бота.
